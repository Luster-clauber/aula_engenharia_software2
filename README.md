# Pong - Jogo Clássico em Python

Este é o código para o jogo clássico **Pong**, implementado em Python utilizando a biblioteca `pygame`. O jogo consiste em duas raquetes e uma bola, e o objetivo é evitar que a bola ultrapasse a sua raquete e marcar pontos ao fazer a bola passar pela raquete do adversário.

## Requisitos

Antes de rodar o jogo, você precisa instalar a biblioteca `pygame`. Você pode instalá-la utilizando o seguinte comando:

```bash
pip install pygame
